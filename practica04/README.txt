Para poder compilar cualquiera de las clases del paquete shapesSVG debes:
1.- Abrir una términal 
2.- Direcccionar la terminal a donde sea que tengas guardada la carpeta practica04
3.- Posicionarte fuera de la carpeta shapesSVG y main , o en otras palabras nada más ingreasar hasta la carpeta src
4.- escribir el comando     javac ./shapesSVG/nombre de la clase que se desea compilar.java
5.- Listo

Para poder compilar la clase prueba.java (que es la clase que tiene el método main) debes:
1.- seguir los primeros 3 pasos de las instrucciones anteriores
2.- escribir el comando    javac ./main/Prueba.java
3.- listo

Para poder ejecutar Prueba.java debes:
1.- seguir los 3 primeros pasos de las instruciones anteriores
2.- escribir el comando java ./main/Prueba.java
3.- el programa debe imprimir en la terminal un código en formato SVG de las figuras (Punto, linea, rectángulo, triángulo).
